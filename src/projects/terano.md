**Technologies:** Node.js, TypeScript, WebSocket, node-canvas  
**Repository:** [alexanderpaolini/terano](https://github.com/alexanderpaolini/terano)

A multifunctional Discord bot developed for user engagement, featuring WebSocket interactions and creative image manipulation.

### Key Features

- Developed interactive utility commands to enhance user experience within Discord.
- Created a meme generator utilizing node-canvas, allowing users to customize and share memes.
- Integrated WebSocket for real-time communication between the bot and Discord users.

### What I Learned

- Hands-on experience with WebSocket communication and the Discord API.
- Image manipulation and generation using node-canvas.
- Building engaging and interactive experiences for users.
